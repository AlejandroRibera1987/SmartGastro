# SmartGastro 

Proyecto orientado al análisis, diseño y desarrollo de un sistema de gestión para Foodtrucks y empresas de catering.


# Descripción del Proyecto

SmartGastro es un sistema pensado para la administración básica de un Foodtruck o empresa de catering.

El trabajo fue desarrollado aplicando conceptos de:

- Análisis de sistemas
- Diagramación
- Diseño de bases de datos
- Programación en Java
- Simulación de APIs REST
- Consumo de APIs externas

Además del desarrollo del sistema, se realizó toda la documentación técnica correspondiente del proyecto.



# Objetivos del Trabajo

- Analizar una problemática real
- Diseñar la estructura lógica del sistema
- Modelar entidades y procesos
- Crear diagramas técnicos
- Implementar funcionalidades en Java
- Diseñar una estructura base de API REST
- Realizar pruebas utilizando Postman
- Consumir una API externa de clima como ejemplo práctico


# Tecnologías Utilizadas

- Python
- JSON
- API REST
- Postman
- Git & GitHub
- Lucidchart
